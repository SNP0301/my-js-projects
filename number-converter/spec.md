### What is Number Converter?

#### 2,8,10,16 진법에 대해 12가지 변환을 지원

##### from Binary

- (bin, oct)
- (bin, dec)
- (bin, hex)

##### from Octal

- (oct, bin)
- (oct, dec)
- (oct, hex)

##### from Decimal

- (dec, bin)
- (dec, oct)
- (dec, hex)

##### from Hexadecimal

- (hex, bin)
- (hex, oct)
- (hex, dec)

### 기능 구성

#### input

- `from_type`: [bin, oct, dec, hex]
- `to_type`: [bin, oct, dec, hex]
- `from_number`: **from_type에 맞는 형식의 정수**

#### output

- ` to_number``:  `from_number`를 `from_type`, `to_type`에 맞춰서 변환한 정수
